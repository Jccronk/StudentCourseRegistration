\doxysubsection{Detailed Description}
Tests the \doxylink{class_course_registration}{Course\+Registration} class. 

Prompts the user to enter the number of course registrations and then collects information for each. It creates \doxylink{class_course_registration}{Course\+Registration} objects based on user input and writes the registration data to a file named \textquotesingle{}course\+\_\+registration\+\_\+data.\+txt\textquotesingle{}. The user can end the input process for course IDs by pressing Enter without typing.
